# 🎓 MegaLearnBot

בוט טלגרם ללימוד אנגלית ומתמטיקה עם AI.

## פיצ'רים
- 🔤 אנגלית: דקדוק, כתיבה, קריאה, דיבור, אוצר מילים
- 🔢 מתמטיקה: חשבון, אלגברה, גאומטריה, סטטיסטיקה, חדו"א
- 🎤 זיהוי קול + משוב על הגייה
- 🔊 תגובות קוליות (ElevenLabs)
- 📊 מעקב התקדמות + XP + רמות
- 🎯 מבחן מיקום אוטומטי
- 🔥 אתגר יומי

## Deploy על Railway

1. צור חשבון ב-railway.app
2. לחץ "New Project" → "Deploy from GitHub repo"
3. חבר את ה-repo הזה
4. Railway יזהה את ה-Procfile ויריץ אוטומטית

## קבצים חשובים
- `main.py` — נקודת הכניסה
- `config.py` — כל המפתחות והגדרות
- `database.py` — ניהול משתמשים (SQLite)
- `ai_brain.py` — ממשק ל-Gemini
- `handlers/` — כל ה-handlers של הבוט
